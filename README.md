To run this website, you must first install XAMPP link: https://www.apachefriends.org/

  When XAMPP installed you need to:
    - Start Apache and MySql
    - Import opac_db at http://localhost/phpmyadmin or click on the "Admin" tab at MySql in XAMPP
        To import the db:
          - Create a db and name it opac_db
          - After the creation of opac_db click on the database and then click "Import"
          - Choose the opac_db file and then scroll down to import
